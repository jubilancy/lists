## Integrations
Integrations empower Teamup to seamlessly connect with other systems, enhancing functionality, streamlining workflows, and improving collaboration. With more integrations on the way, Teamup will continue to expand its capabilities, helping you work more efficiently.
#### API
The Teamup API enables seamless interaction between external systems and the calendar.
#### Webhooks
Webhooks can be used to receive alerts when something changes in the calendar.
#### Embed Wizard
The embed wizard generates URLs and iframe code, making it easy to embed Teamup in your web pages.
