**TL;DR:** [Smoke Signal Hill Call](https://smoke.tel/) is a multi-language, web-based communication platform offering free, peer-to-peer video conferencing, encrypted messaging, and a suite of privacy-focused "gimmick" tools like emoji encryption and secret sharing.

-----

## **Core Features**

  * **[Smoke Signs Call](https://www.google.com/search?q=https://smoke.tel/%23hill):** A 100% peer-to-peer video conferencing tool that runs directly in the browser.
      * No registration or downloads required.
      * Users join via a specific "hill" link (identifier).
      * Optional end-to-end encryption via local password protection.
  * **[Gossip Box](https://www.google.com/search?q=https://smoke.tel/%23gossip):** A public/private chat area for coordination or leaving messages without registration.
  * **[SmileyCrypt](https://www.google.com/search?q=https://smoke.tel/%23smileycrypt):** A playful encryption tool that converts text into strings of emojis. It allows users to hide messages in plain sight, appearing "infantile" to those without the password while securing the actual content.

-----

## **Privacy & Encryption Tools**

  * **[Secret Sharing](https://smoke.tel/share/):** Generates links for sensitive text (passwords, bank info) that delete from the server immediately after the first retrieval.
  * **[Shamir's Secret Sharing (s4)](https://packages.frdl.de/raw/vendor-patch/shamir-secret-sharing-s4/):** A cryptographic algorithm that splits a secret into multiple parts; the secret can only be reconstructed if a specific number of "shares" are combined.
  * **Third-Party Recommendations:** Links to external secure apps like [Signal](https://signal.org/), [Telegram](https://telegram.org/), and [qTox](https://qtox.github.io/en.html).

-----

## **Social & Web Utilities**

  * **[STARTforum](https://startforum.de):** A space to create or join private/public social networks.
  * **[Fediverse City](https://inne.city):** A gateway to decentralized networks like Mastodon, BlueSky, and Nextcloud.
  * **[Webfan Business Card](https://webfan.website):** A central "About Me" page generator to collect social media and project links.
  * **Additional Gadgets:**
      * **[Feed Aggregator](https://webfan.de/apps/feeds/):** Tools to proxy or merge RSS feeds.
      * **[Markdown Editor](https://packages.frdl.de/raw/frdl/markdown-it-micro/):** A simple editor for web formatting.
      * **[Minigames](https://webfan.de/apps/minigames/):** Basic browser-based entertainment.
