Stats, charts and other data visualization:  
[Lastfmstats.com](<https://lastfmstats.com/>)

- Very powerful and the most popular stat tool
- Custom date ranges
- Artists and scrobble count filters
- Tons of different charts
- Metrics for artists, albums, tracks and scrobbles
- CSV and JASON exports for faster loading times

[ScrobbleFlow](<https://westbrooke117.github.io/scrobbleflow>)

- Powerful chart generator
- Artists, albums and tracks
- Different chart styles
- Custom date range
- Custom data set
- Top artists/albums/tracks in selected time must be selected manually

[Last.fm Time Charts](<https://pmcdonough8133.github.io/last.timer/>)

- Table with playtime and playcount data
- Artists and tracks
- Predetermined date range options

[Last history](<https://anhuynh.github.io/last-history/>)

- Cumulative scrobbles chart
- Artists, tracks and albums
- Custom date range
- Custom data set

[Last Chart](<http://www.lastchart.com.s3-website-us-east-1.amazonaws.com/>)

- Different data visualizations - bubble, map, list, cloud, etc.
- Artists only
- Overall or last week date ranges

[Last.fm Tools](<https://nroutasuo.github.io/lastfm-vis/#tagtimeline>)

- Tag timeline, tag cloud and artist timeline
- Album charter showing which albums you listen a lot to or missed
- Predetermined date range options

[Scatter.FM](<https://scatterfm.markhansen.co.nz/>)

- Entire listening history in one graph
- Doesn't work if you scrobble while using it

[Nicholast.fm](<http://nicholast.fm/>)

- Monthly top, yearly top, artists and track recommendations
- Predetermined date range options

[Last.fm Mainstream Factor](<https://mainstream.ghan.nl/>)

- Shows how mainstream your music is
- Predetermined date ranges

[Explr.fm](<https://mold.github.io/explr/>)

- Your artists on map
- Many with missing locations

[Cartogrify](<https://cartogrify.web.app/>)

- Similar to [Explr.fm](<http://explr.fm/>) but only for top artists

[Lastfmviz](<https://lastfmviz.netlify.app/>)

- Every single scrobble visualized by album image on infinitely scrolling page
- Missing album arts ruin it a little bit

[Timewhizzs](<https://timewhizzs.net/>)

- Simple chart of minutes listened to artists
- Predetermined date ranges

[Lasttime](<https://undefinist.com/lasttime/>)

- Minutes listened to artists
- Custom start date
- Requires [Last.fm](<http://last.fm/>) API key (can be generated for free)
- Very slow

[MusiPie](<https://musipie.skybly.net/>)

- Pie chart of your scrobbles

[Bijou.fm stats](<https://www.bijou.fm/#cta>)

- Top artist, tracks and albums
- Custom and predetermined date ranges

[Last.fm taste overlap](<https://7x11x13.xyz/last-fm-overlap/>)

- How do tastes of two users overlap

[Tasty&nbsp;Overlap&nbsp;for&nbsp;Last.fm](<https://app.detrave.net/tasty-overlap/thlayli_x/>)

- How do tastes of two users overlap
- More detailed, more options, graphs

[LastFM Top Artists](<https://davorg.dev/last-live/>)

- Lists your top artist and allows you to select those you've seen live. Displays result in a graph and table
- Predetermined date ranges
- Custom artists count

[Last Live](<https://davorg.dev/last-live/>)

- Generates list of top artists (custom count) and lets you select those you've seen live
- Generates simple graph

[Last.fm Visualizer](<https://v0.hasanjws.com/>)

- Visualization tool in modern design
- Basic overview
- Music discovery
- Listening timeline
- Music evolution
- Artwork mosaic
- Listening Heatmap

[Shikaka Stats](<https://lastfm.shikaka.net/>)

- Very unique site
- Various stats
- Works in interesting and special way (best to see for yourself to understand)
- Has it's own subreddit - check out [r/ShikakaStats](<https://www.reddit.com/r/ShikakaStats/>)

[Finale](<https://web.finale.app/>)

- Comprehensive [Last.fm](<http://last.fm/>) tool
- Predetermined and custom date ranges
- Artists, albums and tracks
- Weekly charts
- Grid and list view
- Line chart for years, months etc.
- Also includes scrobbler, really good collage generator and random song picker
- Requires [Last.fm](<http://last.fm/>) login

[Music Grapher](<https://www.musicgrapher.com/>)

- View connections between artists
- Option to import custom artists list

[Zero Charts](<https://zerocharts.com.br/>)

- View your stats as weekly music charts

Collage generators  
[Last.fm collage generator](<https://www.tapmusic.net/>)

- Albums only
- 3x3, 4x4, 5x5 and 10x10 sizes
- Predetermined date ranges
- Album, artists and playcount display options
- Missing album arts shows as black blanks
- Also has [backup site](<https://www.nsfcd.com/lastfm/>)

[Last.fm top albums patchwork generator](<https://lastfmtopalbums.dinduks.com/>)

- Albums only
- Custom size - 1-20 options for both rows and columns
- Predetermined date ranges
- Custom image size in pixels
- Skips albums with missing artwork
- No border option currently not working

[Monster Shop Collage generator](<https://collage.monstershoprock.com/>)

- Grid and collage styles
- Predetermined date ranges
- Custom columns size for grid option, 2-10
- Was able to fetch album arts others were not
- Sharpen option

[SongStitch](<https://songstitch.art/>)

- Very powerful collage generator
- Albums, artists and tracks
- Predetermined date ranges
- Custom grid size, up to 20x20
- Artist name/Album name/Playcount
- Optional advanced options
    - Grayscale
    - Text size, position and bold option
    - WebP compression

[Apwhite.dev](<https://apwhite.dev/lastfm/>)

- Site says it's deprecated but works just fine
- Albums or artists
- Predetermined date ranges
- Custom grid size, up to 14x14
- Album name
- Option to hide missing artwork

[MusicBucket](<https://musicbucket.net/collage/>)

- Albums, artists and tracks
- Predetermined date ranges
- Custom grid size, up to 5x5

[Chart My Music](<https://www.chartmymusic.com/>) - rainbow collage and picture to collage

- Rainbow Collage
    - Collage sorted by color, brightness or play count
    - Albums only
    - Predetermined date ranges
    - Custom grid size, up to 20x20
    - Play count, names and inverted order options

- Picture to Collage
    - Recreates any uploaded picture with your albums
    - Predetermined date ranges
    - Custom number of top albums, up to 100
    - Custom number of images in collage, up to 20 000
    - Custom opacity option

[Dbeley Last.fm tools](<https://lastfm-tools.dbeley.ovh/>) - collage and wordcloud generator

- Collage generator
    - Only albums
    - Predetermined date ranges
    - Custom grid size, up to 30x30
    - Top100 special collage mode

- Wordcloud generator
    - Wordcloud from artists tags
    - Predetermined date ranges
    - Custom artists count
    - Custom number of top tags

[Lastcollage.io](<https://lastcollage.io/>)

- Only albums (options for artists and tracks are present but don't work, let me know if it gets fixed)
- Custom and predetermined date ranges
- Custom grid size, up to 20x20
- Album and artists name option
- Hide missing album art option

[LastWave](<https://savas.ca/lastwave/#/>)

- Artist, albums or tracks
- Wave visualization
- Custom date range
- Group by day, week, month or year
- Custom minimum plays option
- Custom graph width and height
- Custom graph type and font
- Many other options

[Last.fm Tools](<https://lastfm.dontdrinkandroot.net/>) (different site)

- Artists cloud generator
- Predetermined date ranges

[Tag cloud generator](<https://tagcloud.rainosullivan.com/>)

- Tags and artists cloud generator
- Predetermined date ranges
- Tag filter option

[Last.fm Timeline](<https://westbrooke117.github.io/Last.fm-Timeline/>)

- Timeline consisting of 4x4 collage of top albums for each month
- Calendar with TOP5 in each month at the end

[Musicorum](<https://musicorumapp.com/generate>)

- Collage generator
- Predetermined date ranges
- Predetermined grid size - up to 20x20
- Other data visualizations - tops, duotone, darkly
- Sometimes takes a couple of tries to successfully generate image

Scrobblers  
[Semi-automatic Last.fm scrobbler](<https://scrobbler.jankuca.com/>)

- Manual scrobbling
- Scrobble along another user
- YouTube scrobbling

[The Universal Scrobbler](<https://universalscrobbler.com/>)

- Manual scrobbling
- Manual bulk scrobbling from CSV file
- Database scrobbling
    - [Last.fm](<http://last.fm/>), Discogs and MusicBrainz databases

- Radio scrobbling - only a few stations available
- Scrobble along another user
- Free plan limited to 20 scrobbles/day

[Vinyl Scrobbler](<https://vinylscrobbler.com/>)

- Large database
- Allows importing records from Discogs
- Only scrobbles entire albums
- Custom playtime option
- Free plan limited to 5 albums scrobbled/month

[Open Scrobbler](<https://openscrobbler.com/>)

- Manual scrobbling
- Album scrobbling from [Last.fm](<http://last.fm/>) and Discogs databases
- Scrobble along another user
- Free

[Code Scrobble](<https://codescrobble.com/>)

- Scan barcode of your physical media and scrobble them
- Also support searching, including searching by manually typing barcodes.
- Free

[Bijou.fm Manual Scrobbler](<https://www.bijou.fm/manual-scrobbler>)

- Database and manual scrobbling
- Only scrobbles tracks, not full albums
- Past and future scrobbling
- Custom starting point
- Bulk scrobbling
- Free

[ScrobbleRadio](<https://scrobblerad.io/>)

- Listen to radio station and srobble what they play
- Currently supports radio stations from USA, UK, Canada, Australia, Ireland, Switzerland and France

[Scrobblify](<https://savas.ca/scrobblify/>)

- Great for new [Last.fm](<http://last.fm/>) users
- Export your Spotify data and use them to backfill last two weeks of scrobbling
- Note: two weeks is limit set by [Last.fm](<http://last.fm/>)

Playlist management  
[Spotlistr](<https://www.spotlistr.com/>)

- Generates Spotify playlists only
- Can take different types of data from your [last.fm](<http://last.fm/>) profile or [last.fm](<http://last.fm/>) database
    - Top track from period, loved tracks, similar tracks, etc.

- Other tools not related to [last.fm](<http://last.fm/>)
    - Playlist generation from other sources, playlist cover generator, etc

- Also has grid creator
    - Predetermined date ranges
    - Grid size 3x3 to 10x10
    - Artists, album, playcount and metadata options

[TuneMyMusic&nbsp;](<https://www.tunemymusic.com/cs>)and [Soundizz](<https://soundiiz.com/transfer-playlist-and-favorites>)

- Transfer playlists between different services
- Support for Spotify, Apple Music, Amazon Music, Deezer, Tidal, SoundCloud and many others

[Dbeley Last.fm tools](<https://lastfm-tools.dbeley.ovh/>) - playlist generator

- Generates playlist in text format
- Artist, track, playcount
- Predetermined date ranges
- Custom playlist size, up to 100
- Option to export to CSV
- Option to only include favorite tracks

Games  
[Lastguess](<https://lastguess.net/>)

- Guess the title, artist and next line of your top songs based a single lyric line

[Album Hangman](<https://album-hangman.surge.sh/>)

- Guess album based on blurred album art
- Works like popular game Hangman

[AlbumIdentify](<https://westbrooke117.github.io/AlbumIdentify/>)

- Guess the name of an album and the artists by slowly uncovering album art
- Predetermined date ranges
- Four difficulty options

Others

[Descent.live](<https://descent.live/>)

[Dbeley Last.fm tools](<https://lastfm-tools.dbeley.ovh/>) - scraping tool

[Lasthop](<https://lasthop.holidaypuzzles.co.za/>)

[Instafest](<https://www.instafest.app/>)

[Last.fm Iceberg](<https://lastfm-iceberg.dawdle.space/>)

[Nixinova Last.fm Tags Tools](<https://www.nixinova.com/tools/lastfm/tags>)

[Jake Ledoux Live](<https://jakeledoux.com/live/>)

- Grid view of what are the people you follow listening to right now or what they have been listening to the last time

[Listening Facts](<https://listeningfacts.com/login>)

- View your scrobbles in form resembling nutrition facts label.

Programs and scripts  
*Note: these are not sites. They need to be downloaded and run as either usual PC program or browser script from extension such as Tempermonkey*  
[Bleh](<https://katelyn.moe/bleh/>)

- Redesign of [Last.fm](<http://last.fm/>) site using script and theming
- Custom color theme, redesigned gallery, profile and library, community-contributed music title capitalization and more
- Tempermonkey or similar browser extension is needed

[Last.fm Scrubbler WPF](<https://github.com/SHOEGAZEssb/Last.fm-Scrubbler-WPF?tab=readme-ov-file>)

- Great and powerful scrobbler
- Manual scrobbler, friend scrobbler, database scrobbler and many others
- Also includes extra functions such as paste your taste, collage generator, milestone checker and others
- Only works on Windows

[Last.fm Extra Stats](<https://c26k.com/lastfmextrastats/>)

- Very powerful tool for data visualization
- Charts, wave graph, line graph, tag cloud and many others
- Date range selection is little bit clunky
- Only works on Windows

[Last.fm Bulk Edit](<https://github.com/RudeySH/lastfm-bulk-edit>)

- Popular script for mass editing scrobbles
- Requires Tempermonkey (or similar script manager) and [Last.fm](<http://last.fm/>) Pro
